Make Aliases for Hugo Posts Pages and Github Push

Wednesday, Dec 11, 2024 | 1 minute read | Updated at Wednesday, Dec 11, 2024

Satish Iyer

Certain aliases to make our hugo posts, pages and github push easier

# aliases for hugo
hpost() {
   hugo new content content/posts/"$1".md
   nvim content/posts/"$1".md
}

hpage() {
   hugo new content pages/"$1".md
   nvim content/pages/"$1".md

}

hugopush() {
   cd public
   git add .
   git commit -m "$1"
   git push origin main
}
